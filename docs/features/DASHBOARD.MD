# Dashboard & Reporting

The SaleSpider dashboard provides a comprehensive overview of your store's performance with real-time insights and analytics.

![Dashboard Overview](/images/overview.png)

## Overview

The dashboard is the central hub of SaleSpider, offering role-based views tailored to different user types:

- **Admin Dashboard**: Complete system overview with all metrics
- **Manager Dashboard**: Store performance and staff management
- **Cashier Dashboard**: Simplified view focused on daily operations

## Key Features

### Real-Time Metrics

**Sales Overview**

- Today's sales total
- Sales count
- Average transaction value
- Weekly and monthly totals

**Inventory Status**

- Total products
- Low stock alerts
- Out of stock items
- Inventory value

### Visual Analytics

**Sales Charts**

- Daily sales (last 7 days)
- Weekly comparison (this week vs last week)
- Monthly sales (last 6 months)

### Quick Actions

From the dashboard, you can quickly:

- Record a new sale
- Add products
- View low stock items
- Access reports
- Manage staff

## Dashboard Components

### Sales Summary Card

Displays key sales metrics with visual charts showing daily, weekly, and monthly trends.

### Low Stock Alerts

Real-time notifications for products below threshold with quick access to inventory management.

### Recent Sales

List of recent transactions:

- Transaction ID
- Total amount
- Cashier name
- Date
- Status

## Troubleshooting

### Dashboard Not Loading

1. Check internet connection
2. Clear browser cache
3. Verify user permissions
4. Check server status

### Data Not Updating

1. Refresh the page
2. Check database connection
3. Verify background jobs
4. Review error logs

### Slow Performance

1. Clear old data
2. Optimize database
3. Check server resources
4. Reduce displayed metrics
